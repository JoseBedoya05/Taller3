## Cómo correr la app de Streamlit incluida

La app incluida es un punto de partida para mostrar:
- recuperación de chunks,
- pregunta del usuario,
- respuesta con contexto,
- revisión rápida de resultados guardados.

### Estructura sugerida del proyecto en GitHub

```text
repo/
├── app_streamlit.py
├── requirements_streamlit.txt
├── data/
│   ├── chunks.parquet
│   ├── comparacion_detallada.csv
│   └── comparacion_por_tarea.csv
└── README.md
```
